# Google SERP Exporter

Praktický test pro Inizio.

## Technologie

- PHP 8.3
- Nette Framework
- SQLite
- PHPUnit
- Docker Compose

## Funkce

- Zadání klíčového slova
- Získání organických výsledků z první stránky Google
- Export do JSON
- Uložení výsledků do SQLite

## Spuštění

```bash
composer install
php -S localhost:8000 -t www
